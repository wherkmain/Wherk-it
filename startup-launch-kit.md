# Startup Launch Kit

*Reusable checklist for launching digital products/services. Skip the legal/foundation stuff Lee already has covered.*

## Phase 1: Foundation (Days 1-3)

### Domain & Web Presence
- [ ] Domain procurement (check availability, secure primary + variations)
- [ ] DNS setup (Cloudflare recommended)
- [ ] Hosting environment configured
- [ ] SSL certificate active

### Core Pages
- [ ] Landing page / home
- [ ] About page
- [ ] Contact form (with spam protection)
- [ ] Privacy policy (generated + customized)
- [ ] Terms of service (generated + customized)
- [ ] 404 page

### Brand Assets
- [ ] Logo (primary + variants)
- [ ] Favicon
- [ ] Color palette defined
- [ ] Typography system
- [ ] Social media avatar (consistent across platforms)

---

## Phase 2: Social Infrastructure (Days 4-7)

### Platform Presence
- [ ] Twitter/X handle secured
- [ ] LinkedIn company page
- [ ] YouTube channel (if video content planned)
- [ ] Instagram (if visual product)
- [ ] TikTok (if applicable)
- [ ] Discord server (if community-focused)

### Profile Optimization
- [ ] Consistent bio/description across platforms
- [ ] Link in bio (Linktree or direct)
- [ ] Branded headers/covers

---

## Phase 3: Content & Automation (Week 2)

### Content Systems
- [ ] Blog/CMS setup
- [ ] Email capture (newsletter signup)
- [ ] Email service provider connected (ConvertKit/Mailchimp)
- [ ] Welcome sequence drafted

### Social Automation
- [ ] Scheduling tool connected (Buffer/Hypefury)
- [ ] Content calendar template
- [ ] Auto-crosspost rules
- [ ] Engagement monitoring setup

### Analytics
- [ ] Google Analytics 4
- [ ] Search Console
- [ ] Social pixel tracking
- [ ] Basic conversion goals defined

---

## Phase 4: Launch Mechanics (Week 3)

### Pre-Launch
- [ ] Waitlist/early access page
- [ ] Beta tester recruitment
- [ ] Launch announcement content drafted
- [ ] Press kit/assets prepared

### Launch Day
- [ ] Social post schedule
- [ ] Email blast queued
- [ ] Communities notified (Reddit, IndieHackers, etc.)
- [ ] Support channel ready

---

## Phase 5: Post-Launch (Ongoing)

### Iteration
- [ ] User feedback loop
- [ ] A/B testing framework
- [ ] SEO content pipeline
- [ ] Retargeting ads (if budget allows)

---

1. Landing page template (Webflow/Framer/Next.js)

### Automations

### Scripts/Tools
